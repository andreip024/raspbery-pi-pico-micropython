# MH-Z19B Micropython and pinout for Raspberry Pi Pico W

## CO2 sensor

### Outputs

- CO2 / ppm

### Install prerequisites

- Micropython (tested with v1.20.0)

### Pinout

Vin -> VBUS (red)  
GND -> GND (black)  
Tx -> GP4 (yellow)  
Rx -> GP5 (blue)
