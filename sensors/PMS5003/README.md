# PMS5003 Micropython and pinout for Raspberry Pi Pico W

## Particulate Matter Sensor

### Outputs

 - PM 1.0 standard
 - PM 2.5 standard
 - PM 10 standard
 - PM 1.0 atmospheric
 - PM 2.5 atmospheric
 - PM 10 atmospheric
 - UM 0.3
 - UM 0.5
 - UM 1.0
 - UM 2.5
 - UM 5.0
 - UM 10

### Install prerequisites

- Micropython (tested with v1.20.0)
- pms5003-micropython (tested with v0.0.7) 

### Pinout

Vin -> VBUS (red)  
GND -> GND (black)  
SET -> GP3 (grey)  
TX  -> GP9 (yellow)  
RX  -> GP8 (blue)  
RESET -> GP4 (white)
