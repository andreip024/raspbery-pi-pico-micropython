# MQ-2 Micropython and pinout for Raspberry Pi Pico W

## Gas/Smoke Sensor

### Outputs

- Smoke / ppm
- LGP (Liquefied petroleum gas) / ppm
- Methane / ppm
- Hydrogen / ppm 

### Install prerequisites

- Micropython (tested with v1.20.0)

### Pinout

Vin -> VBUS (red)  
GND -> GND (black)  
A0 -> GP26 (purple)
